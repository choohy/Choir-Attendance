# Choir Attendance

Simple web app to track choir attendance, save to a checked-in JSON file, and export CSV by date.

Run locally:

```bash
npm install
npm start
# open http://localhost:3000
```

File storage:

- `data/attendance.json` stores members and attendance records.

Features:

 - Import a CSV of member names before taking attendance
 - Add per-event guest records (name, email, phone) and include them in exports
